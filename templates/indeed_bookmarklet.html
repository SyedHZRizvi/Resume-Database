How to use
1
Show your bookmarks bar. In Chrome / Edge press Ctrl+Shift+B (Windows) or Cmd+Shift+B (Mac). Firefox: View → Toolbars → Bookmarks Toolbar.
2
Drag the green “Save to TransCrypts” button above into your bookmarks bar. It will appear as a new bookmark.
3
Open Indeed Resume Search and view any candidate's profile.
4
Click the bookmark. A small preview shows the parsed fields (name / email / phone / position / resume URL). Review them, then click Save. A confirmation toast appears in the bottom-right corner.
5
Skip the preview by holding Shift while clicking the bookmark — the import runs immediately. Useful when you're sweeping a long candidate list.

Indeed's class names change frequently, so the bookmarklet uses generic queries (page title, mailto/tel links, body text scans). If a field comes back blank, edit the candidate in TransCrypts to fix it — the import still succeeds.
